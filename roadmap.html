Create Mod Factory Builder

The Official Development Roadmap to Version 2.0

Phase 1: Foundational Enhancements & UX

The focus of this phase is to refine the core user experience, improve visual feedback, and add essential quality-of-life features that make the tool more intuitive and powerful for everyday use.

v1.1

The Quality of Life Update

  • Multi-Selection & Grouping: Implement box selection and allow users to group nodes (Ctrl+G) to be moved together.
  • Enhanced Conveyor Visuals: Add animated textures to conveyors for direction and introduce distinct styles for belts, funnels, and chutes.
  • Snapping Grid Improvements: Add a toggleable visible grid and allow users to adjust its size for precise placement.
  • Basic Add-On Support: Add custom Machine nodes not covered by base create.
v1.2

The Organization Update

  • Layers & Z-axis: Introduce a layer management system for building multi-story factories by moving nodes up or down.
  • Labeling and Annotations: Allow users to add text labels directly onto the canvas to describe parts of their factory.
  • Project File Management: Implement autosave to local storage and improve import/export with better error handling.

Phase 2: Introducing Logic & Power

This phase focuses on integrating the core mechanics of the Create Mod, starting with the power system, Redstone logic, and fluid management.

v1.3

The Power Update

  • Rotational Power: Add nodes for power sources (Water Wheels, etc.) and transfer components (Shafts, Cogs).
  • Power Network Simulation: Implement a basic Stress Unit (SU) calculation to show power generation vs. consumption.
v1.4

Smarter Factories

  • Redstone & Logic: Add nodes for Redstone and Create-specific logic components like Content Observers.
  • Basic Logic Simulation: Link logic components to machines to visualize their "on/off" state.
v1.5

The Fluids Update

  • Fluid Management: Introduce nodes for Pumps and Tanks, with Pipes as a new connection type.
  • Fluid Network Visualization: Allow users to specify fluid types and visualize flow.

Phase 3: Automation & Full Feature Parity

This phase aims to bring the planner closer to the full capabilities of the Create Mod, focusing on complex automation with trains and contraptions, and culminating in the ultimate planner experience.

v1.6

The Railway Update

  • Train and Track Planning: Add tools for designing train tracks with curves and slopes.
  • Train Components: Introduce nodes for Stations, Signals, and Train Controls, and allow train assembly.
v1.7

The Contraptions Update

  • Movable Contraptions: Introduce components like Bearings, Gantries, and Rope Pulleys.
  • Contraption Mode: Add a special mode to assemble components onto a chassis and visualize movement.
v2.0

The Ultimate Factory Planner

  • Full Recipe Database: Integrate a complete database of Create Mod item recipes.
  • Calculation and Analytics: Provide a dashboard that calculates total resource input and item output per minute.
  • UI Overhaul: A complete user interface refresh based on a year of user feedback.

Post-v2.0 Wishlist: Advanced Features & Collaboration

These are ambitious features that are planned for development after the core v2.0 roadmap is complete. Their implementation will depend on community feedback and technical feasibility.

Blueprint & Schematic Support

Blueprint Creation

  • Allow users to select a group of machines and save them as a reusable "blueprint."
  • Create a blueprint library where users can store and manage their custom blueprints.

Schematic Export (Major Feature)

  • Export the factory plan as a `.nbt` schematic file, usable in-game with the Schematicannon.
  • This will bridge the gap between planning and in-game execution.
